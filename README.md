# Cyber Decryptor - Professional Ransomware Recovery Website

A modern, professional website for ransomware recovery services built with HTML, CSS, and JavaScript. This website provides comprehensive information about ransomware recovery services, emergency contact options, and interactive tools for victims of ransomware attacks.

## 🚀 Features

### Core Features
- **Responsive Design**: Fully responsive layout that works on all devices
- **Emergency Contact System**: 24/7 emergency hotline with prominent call-to-action
- **Ransomware Identification Tool**: Interactive file upload tool for ransomware analysis
- **Professional Services**: Comprehensive service offerings including recovery, consulting, and prevention
- **Trust Indicators**: Certifications, testimonials, and success statistics
- **Emergency Form**: Priority-based contact form with immediate response system

### Interactive Elements
- **File Upload & Analysis**: Drag-and-drop ransomware identification tool
- **Live Chat Widget**: Emergency support chat functionality
- **Mobile Navigation**: Hamburger menu for mobile devices
- **Smooth Scrolling**: Enhanced navigation experience
- **Animated Statistics**: Counter animations for impact metrics
- **Emergency Popup**: Critical alert system for urgent cases
- **Notification System**: Real-time feedback for user actions

### Design Highlights
- **Modern UI/UX**: Clean, professional design inspired by industry leaders
- **Accessibility**: Focus states, keyboard navigation, and semantic HTML
- **Performance**: Optimized CSS and JavaScript for fast loading
- **Security Theme**: Professional cybersecurity aesthetic with blue/red color scheme
- **Trust Building**: Certifications, testimonials, and success metrics prominently displayed

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Web server (optional, for local development)

### Installation

1. **Clone or download the project files**
2. **Open in a web browser**:
   - Double-click `index.html` to open directly in browser
   - Or serve via local web server for full functionality

### Local Development Server (Optional)

For full functionality, especially file uploads, run a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (with http-server package)
npx http-server

# Using PHP
php -S localhost:8000
```

Then visit `http://localhost:8000` in your browser.
